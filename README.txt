================================================================
  台股預測分析系統 v1.4.0
  Taiwan Stock Prediction & Analysis Desktop App
================================================================

【系統需求】
  - Windows 10 / 11 (64-bit)
  - 網路連線（用於下載股價資料與新聞）

【使用說明（一般使用者）】
  解壓縮後，直接點兩下「台股預測分析系統.exe」即可啟動。
  不需要安裝 Python 或任何其他軟體。

  1. 在頂部輸入框填入股票代號
     常見台股代號範例：
       0050.TW  - 元大台灣50 ETF
       2330.TW  - 台積電
       2317.TW  - 鴻海
       2454.TW  - 聯發科
       006208.TW - 富邦台50
       0056.TW  - 元大高股息

  2. 點擊「預測明天」按鈕
     ※ 首次使用需訓練 Transformer 模型，約需 4~5 分鐘（視電腦效能）
     ※ 訓練完成後會自動儲存，下次載入只需幾秒

  3. 查看預測結果
     - K 線圖：顯示近期走勢 + MA5/MA20 均線
     - 右側面板：顯示明日上漲/下跌機率
     - 信心度：直覺呈現模型把握程度（高/中/低）
     - 主要驅動因子：SHAP 解析影響預測的關鍵指標
     - 新聞情緒分析：AI 自動搜尋近期新聞並分析正負面
     - 未來 3 日走勢推估

  4. 自動更新
     啟動時自動檢查新版本，點「立即更新」一鍵完成。

【開發者說明】
  啟動：python main.py
  打包：python build.py
  依賴：pip install -r requirements.txt

【核心技術】
  - Transformer（3 層 Encoder，300 天窗口）— 業界主流時序模型
  - LightGBM Ensemble（3 模型投票）— 梯度提升樹分類
  - OpenAI GPT — 新聞情緒分析 + 3 日走勢推估
  - Brave Search API — 即時新聞搜尋

【注意事項】
  ⚠️ 本軟體僅供學術研究與參考使用
  ⚠️ 股票市場存在不可預測的風險
  ⚠️ 預測結果不構成任何投資建議
  ⚠️ 請勿將本軟體的輸出作為實際交易決策依據

【資料存放位置】
  使用者資料：%LOCALAPPDATA%\台股預測分析系統\
  程式日誌：%LOCALAPPDATA%\台股預測分析系統\logs\
