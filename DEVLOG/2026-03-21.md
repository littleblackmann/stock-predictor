### 2. 股票代碼智慧輸入（SmartLineEdit）

**新增 `data/tw_stock_list.py`**
- 硬編碼約 150 支台股代碼 → 中文名稱對照表（ETF、半導體、金融、傳產、電信、零售）
- 涵蓋最新 ETF（00946～00967）

**新增 `data/stock_fetcher.py`**
- 從 TWSE/TPEX ISIN 頁面動態抓取全市場所有股票代號（Big5 解碼 + U+3000 正則解析）
- 快取至 `stock_list_cache.json`，每 7 天自動更新
- `get_stock_dict(fallback)`：合併快取 + 硬編碼清單
- `needs_refresh()`：判斷是否需要背景更新
- `refresh_cache()`：重新抓取並覆寫快取

**新增 `ui/smart_line_edit.py`**
- `SmartLineEdit(QLineEdit)`：股票代碼智慧輸入框
- 輸入任何字元即觸發搜尋，同時比對代碼與中文名稱
- ↑↓ 方向鍵瀏覽，Enter 確認，Esc 關閉，點擊選擇
- Popup 使用 `Tool | FramelessWindowHint | WindowStaysOnTopHint`，不搶焦點
- `_reposition_popup`：自動偵測螢幕底部邊界，超出時改為向上彈出（修復抽屜底部輸入框 popup 被截斷問題）

**修改 `ui/main_window.py`**
- 使用 SmartLineEdit 取代原本的 QLineEdit
- 啟動後 3 秒（若快取過期）在背景執行緒自動更新股票清單
- `_stock_dict = get_stock_dict(TW_STOCK_LIST)` 合併動態 + 靜態清單

### 3. 預測記錄機制（Prediction Log）

**新增 `data/prediction_logger.py`**
- `PredictionLogger`（靜態方法類別）
- `append(result)`：每次預測後自動寫入一筆（symbol、date、predicted、up_prob、3日走勢、actual 空白）
- `backfill_actuals()`：用 yfinance 回填 actual/actual_return/correct 欄位，依 symbol 分組批次抓取，避免重複請求
- `_format_forecast()`：將 list[dict] 格式化為「明日:偏多(高) / 後天:盤整(中) / +3天:偏空(低)」可讀文字
- `migrate_gpt_3day()`：修復舊記錄中儲存為 Python repr 格式的 3日走勢欄位
- `delete_rows(indices)`：刪除指定 CSV 行
- `get_stats()`：整體 + 各股票準確率統計

**新增 `ui/prediction_log_dialog.py`**
- `PredictionLogDialog(QDialog)`：點「📊 記錄」按鈕開啟
- QTableWidget 顯示全部預測記錄（最新在最上，orig_idx 存於 UserRole）
- 欄位：預測日期、股票、預測方向、上漲機率、3日走勢、實際結果、漲跌%、正確
- 顏色標示：up=綠、down=紅、✓=亮綠、✗=亮紅
- 🗑 刪除選取列（支援多選，QMessageBox 確認）
- 🔄 更新實際結果（BackfillWorker 背景執行緒，完成後自動刷新表格）
- ↓ 匯出 CSV
- 底部顯示整體準確率 + 各股票分別準確率

**修改 `ui/main_window.py`**
- `_on_prediction_finished`：預測完成後自動呼叫 `PredictionLogger.append`
- 啟動後 2 秒在背景自動回填 `prediction_log.csv` 中尚未有 actual 的記錄
- 新增「📊 記錄」按鈕（`#btnLog`）

## Day 3 補充（下午）

### Bug Fix：★ 按鈕無法關閉自選股抽屜
**問題**：按 ★ 只會開啟抽屜，再按一次無法收起
**原因**：`_on_watchlist_add_clicked` 永遠呼叫 `open_drawer()`，且 `open_drawer()` 內部會強制重置 `_is_open = False`
**修正**：改為 toggle 邏輯，`_is_open` 為 True 時呼叫 `close_drawer()`

### 新功能：啟動回填通知
**問題**：backfill 在背景靜默執行，使用者完全不知道有沒有跑、填了幾筆
**修正**：
- Signal 改為 `done = Signal(int, int, int)`（filled, correct, total）
- 回填完成後若有新資料，狀態列顯示：`回填完成：N 筆 ｜ 歷史準確率 X%（Y/Z 筆）`
- 沒有資料可回填時靜默（不干擾）

### UI 主題全面換色：深藍 → 鐵灰
**變更**：
- `ui/styles.qss`：所有深藍背景（`#0B1426` 系列）換成鐵灰（`#1A1A1A` 系列），霓虹色保留

- `ui/prediction_panel.py`：進度條、走勢卡片、分隔線、section title 等 inline style 全部換色
