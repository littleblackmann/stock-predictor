# 台股預測分析系統 — 產品介紹

> **版本：** v1.3.2 ｜ **發佈日期：** 2026 年 3 月 30 日

---

## 這是什麼？

一套安裝在電腦上的 **台股漲跌預測工具**。

輸入任何一支台灣股票代碼（例如 0050、2330），系統就會告訴你：

- **明天大概會漲還是跌？** 機率各是多少？
- **系統有多大把握？** 高 / 中 / 低信心等級
- **為什麼這樣判斷？** 列出影響最大的前 5 個原因
- **最近新聞怎麼說？** AI 自動分析近期新聞的正面或負面程度
- **未來 3 天的走勢推估**

---

## 怎麼用？

```
解壓縮 → 點兩下「台股預測分析系統.exe」→ 輸入股票代碼 → 看結果
```

- 不需要安裝任何東西，解壓就能用
- 支援 Windows 10 和 Windows 11
- 第一次使用會要求輸入 OpenAI API Key（用於 AI 新聞分析）
- 內建自動更新功能，有新版本時一鍵完成更新

## 畫面上會看到什麼？

### K 線走勢圖
就像券商軟體看到的 K 線圖，紅色是漲、綠色是跌，還有 5 日和 20 日均線。可以用滑鼠滾輪縮放、拖曳查看歷史走勢。

### 預測結果
- **大字顯示「上漲」或「下跌」**
- 兩條進度條分別顯示上漲機率和下跌機率（例如：上漲 62%、下跌 38%）
- 信心等級用顏色和文字標示（綠色 = 高信心、黃色 = 中等、橘色 = 低）

### 判斷依據（前 5 名）
系統會列出影響這次預測最大的 5 個因素，例如：
- 「週線 RSI 強弱：看多」— 表示最近一週的技術指標偏強勢
- 「外資連續買超：看多」— 表示外資連續好幾天在買這支股票
- 「VIX 恐慌指數：看空」— 表示美國市場最近比較恐慌

### 預測進度
按下預測後會彈出進度對話框，顯示目前執行到哪個步驟（下載股價 → 抓取籌碼 → 美股資料 → 特徵工程 → Transformer → LightGBM → 新聞分析 → 生成預測），每個步驟都有圖示和百分比，不會再讓人以為程式當機了。

### 新聞情緒
AI 會自動搜尋這支股票最近的新聞，分析整體偏正面還是負面，並標示：
- 情緒分數（-1.0 極度看空 到 +1.0 極度看多）
- 影響程度（高 / 中 / 低）
- 最關鍵的新聞事件是什麼

### 未來 3 日走勢
以卡片方式顯示明天、後天、大後天的趨勢預估，每天都有方向（偏多 / 偏空）和原因說明。

## 它是怎麼預測的？（白話版）

系統的預測分成三個步驟，就像問三個不同的專家然後綜合意見：

### 第一步：看歷史走勢找規律
系統會看這支股票過去 **7 年的歷史資料**（大約 2,500 個交易日），用 **Transformer**（跟 ChatGPT 同類型的 AI 技術）從中學習價格走勢的規律。它一次看 300 天的資料，能發現「每年這個時候通常會怎樣走」的季節性規律，就像一個看了幾千張 K 線圖的老手。

### 第二步：綜合多方面資訊判斷
除了看走勢圖，系統還會參考：

| 參考資訊 | 白話說明 |
|---------|---------|
| 技術指標 | RSI、MACD、布林帶等，就像量體溫一樣量股票的「健康狀態」 |
| 美股隔夜表現 | 昨晚美國股市漲還是跌？通常會影響隔天台股開盤 |
| 三大法人動態 | 外資、投信、自營商是在買還是在賣？大戶的動向很重要 |
| 融資融券 | 散戶借錢買股票（融資）或借股票來賣（融券）的狀況 |
| 籌碼進階分析 | 外資加速度、法人共振、融資股價背離等 13 個衍生指標 |
| 市場氣氛 | 目前是多頭、空頭還是盤整？趨勢強度和波動率也會影響判斷 |

系統會把這些資訊丟給 **3 個獨立的 AI 模型** 分別判斷，然後取平均——就像三個分析師各自給意見，比一個人說了算更可靠。

### 第三步：參考最新新聞微調
最後，AI 會上網搜尋這支股票最近的新聞，判斷整體偏正面還是負面，然後小幅微調預測結果。但新聞的影響最多只佔 15%，避免被聳動標題誤導。

## 它準嗎？

### 客觀數據
- 模型初期準確率約 **52～55%**（略高於擲硬幣的 50%）
- 使用 Transformer 技術，能看到 **300 天的長期規律**（季節性、財報週期等）
- 訓練時自動調低舊資料權重，以最近 2 年的走勢為主
- 隨著持續使用，系統會 **自動學習並進步**（每 7 天自動更新模型）
- 當準確率下降到一定程度，系統會自動觸發重新訓練

### 什麼時候比較準？
- **信心等級「高」的預測** 通常比「低」的更可靠
- **大型權值股**（如台積電、0050 ETF）通常比冷門小型股準
- **正常盤勢** 比突發事件（戰爭、疫情）時更準

### 什麼時候不太準？
- 突發重大事件（系統來不及反應）
- 冷門小型股（歷史資料不夠多）
- 市場極端恐慌或極端樂觀時

## 自選股功能

可以把常看的股票加入「自選股清單」，下次打開就能快速切換，不用每次都打代碼。

- 點右上角的「★」按鈕新增
- 從左側滑出的抽屜快速選擇
- 支援中文搜尋股票名稱（打「台積」就能找到 2330）

## 自動更新

系統內建自動更新功能：
- 每次啟動自動檢查是否有新版本
- 也可以在「設定 → 關於/更新」手動檢查
- 點「立即更新」後全自動完成（下載 → 安裝 → 重啟），不需要手動解壓縮
- 更新只下載有變動的檔案（通常只有幾 MB），不用每次重新下載整包 650 MB
- 你的設定、自選股、模型、預測記錄都不會被覆蓋
