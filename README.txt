================================================================
  台股預測分析系統 v1.0.0
  Taiwan Stock Prediction & Analysis Desktop App
================================================================

【系統需求】
  - Windows 10 / 11 (64-bit)
  - Python 3.10 或以上版本
  - 網路連線（用於下載股價資料）

【安裝步驟】
  1. 安裝相依套件：
     pip install -r requirements.txt

     ※ TensorFlow 安裝說明：
       - 有 GPU：pip install tensorflow[and-cuda]
       - 僅 CPU：pip install tensorflow-cpu

  2. 啟動應用程式：
     python main.py

【使用說明】
  1. 在頂部輸入框填入股票代號
     常見台股代號範例：
       0050.TW  - 元大台灣50 ETF
       2330.TW  - 台積電
       2317.TW  - 鴻海
       2454.TW  - 聯發科
       006208.TW - 富邦台50
       0056.TW  - 元大高股息

  2. 點擊「預測明天」按鈕
     ※ 首次使用需訓練模型，約需 3~10 分鐘（視電腦效能）
     ※ 訓練完成後，下次啟動會自動載入模型（秒速）

  3. 查看預測結果
     - K 線圖：顯示近期走勢 + MA5/MA20 均線
     - 右側面板：顯示明日上漲/下跌機率
     - 信心度進度條：直覺呈現模型把握程度
     - 主要驅動因子：SHAP 解析影響預測的關鍵指標

  4. 匯出報表
     點擊「匯出報表」將預測結果存為 CSV 檔案

  5. 重新訓練
     點擊「重新訓練」可清除舊模型並重新訓練
     （建議每月重訓一次以保持模型新鮮度）

【注意事項】
  ⚠️ 本軟體僅供學術研究與練習目的
  ⚠️ 股票市場存在不可預測的風險
  ⚠️ 預測結果不構成任何投資建議
  ⚠️ 請勿將本軟體的輸出作為實際交易決策依據

【日誌檔案】
  程式執行紀錄存放於：logs/stock_app_YYYYMMDD.log
  若發生問題，請提供此檔案內容協助除錯
