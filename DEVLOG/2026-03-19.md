# Day 1 — 2026-03-19

## 完成項目

### 架構建立
- 從零建立台股預測分析系統 v1.0
- 確立整體技術架構與資料夾分層

### UI（PySide6）
- 深色霓虹主題（藏青底 + 霓虹綠紅），QSS 獨立樣式檔
- 無邊框圓角按鈕含 Hover 動畫
- 圖表下方大型藍色進度列
- 右側預測面板（寬版、大字體）
- 狀態列顯示模型新鮮度（幾天前訓練）

### 圖表
- TradingView Lightweight Charts 嵌入 QWebEngineView
- K線 + MA5（橘）/ MA20（藍）均線
- 成交量副圖（漲跌自動著色）
- 預測箭頭標記在最新K線

### 預測引擎
- `FeatureEngineer`：RSI、MACD、布林通道、ATR、量比等 13 維技術特徵
- `LSTMExtractor`：60日回望，雙層 LSTM(128→64)，萃取時間特徵向量
- `LGBMClassifier`：融合 LSTM 特徵(64維) + 技術特徵(13維) 做二元分類
- SHAP 可解釋性分析（前5大驅動因子）
- 模型自動重訓機制（超過 7 天觸發）

### GPT 分析
- Google News RSS 抓取 10 則台股新聞（免費，不需 API）
- OpenAI GPT 情緒分析，分數 -1.0 ~ +1.0
- 情緒融合公式：最終機率 = ML機率 × 85% + GPT情緒調整 × 15%
- 未來 3 日走勢卡片（趨勢 + 信心 + 原因）

### 其他
- `QThreadPool` 背景執行緒，UI 不卡頓
- `QueueHandler` 非同步日誌寫入
- CSV 報表匯出功能
- `yfinance` 下載約 700 天歷史資料（確保 LSTM 訓練量足夠）

## 技術架構

```
stock_predictor/
├── main.py                  # 入口點
├── config.json              # OpenAI Key + 模型設定
├── ui/
│   ├── main_window.py       # 主視窗，整合所有元件
│   ├── chart_widget.py      # TradingView 圖表
│   ├── prediction_panel.py  # 右側預測面板
│   └── styles.qss           # 深色霓虹主題
├── workers/
│   └── prediction_worker.py # QRunnable 背景任務
├── data/
│   ├── yfinance_adapter.py  # 歷史資料下載
│   └── news_sentiment.py    # GPT 新聞情緒分析
├── features/
│   └── feature_engineer.py  # 技術指標計算
├── models/
│   ├── lstm_extractor.py    # LSTM 特徵萃取器
│   ├── lgbm_classifier.py   # LightGBM 分類器
│   └── saved/               # 訓練好的模型檔案
└── logger/
    └── app_logger.py        # 非同步日誌系統
```
